 OPX Parallax is a mod that seeks to add Parallax Continued support to the full suite of OPX mods. 
 Dependencies are 
 OPX-Jool, OPX-Sarnus, OPX-Urlum, OPX-Neidon, Outer Planets Mod, and Parallax Continued.
 If you find that it runs without any of the former 4, they haven't been added yet as this mod is in heavy WIP.
